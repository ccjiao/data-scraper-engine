# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.0.x   | ✅ |

## Reporting a Vulnerability

If you discover a security vulnerability, please:

1. **Do NOT** open a public issue
2. Email: wxccjiao@gmail.com with subject `[SECURITY] data-scraper-engine`
3. Include: description, steps to reproduce, potential impact
4. We will respond within 48 hours and work on a fix

## Responsible Use

This tool is designed for legitimate data collection. Users must:

- Only collect publicly accessible data
- Respect website terms of service and robots.txt
- Comply with applicable laws (Cybersecurity Law, PIPL, GDPR, etc.)
- Not collect personal private information without consent
- Not use this tool for any illegal or unethical purposes
